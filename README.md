## backtesting 코드
여러 트레이딩 전략들을 직접 백테스팅하여 검증해보고자 만든 프로젝트

백테스팅 라이브러리가 많지만, 다양한 전략을 테스트해보기 위해서는 한계가 있어, 직접 구현함(오류 있을 수 있음)

### 설치 및 실행
`$ python -m venv backtesting`

`$ backtesting\Scripts\activate`

`$ python main.py`
